GIT CONFIG

Configuring git is a global configuration settings live in a file called (.gitconfig) which typically lives in your home directory. To view your present system configuration settings type: cat ~/.gitconfig will display all your command line configurations settings. I will briefly discuss
three functional tags associated with gitconfig:

  • git config --global: This tag is used to set the authors credintials or details.Usually Names and Email which must correspond with email used in registering at github profile creation.
  • git config --global --replace-all: This tag is set to help programmmer control the output page which result fro 'cat' command. It is normally used with 'core.pager'.
  • git alias.(keyboard command): It allows users to create shortcut keyboard command at command prompt rather than just reapeating typing same command again and again.
